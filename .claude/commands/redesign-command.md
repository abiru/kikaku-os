# フロントエンド全面デザインリニューアル コマンド

⚠️ このタスクは他の全Issueより最優先で実行すること。

## 実行手順

1. `tailwindplus/` の全テンプレートを確認し、使えるパターンをリストアップ
2. `apps/storefront/src/` の現在の構成を把握
3. テンプレートをベースに、apple.com / stripe.com / airbnb.com 水準のデザインを実装

## 実装順

1. デザインシステム基盤（tailwind.config拡張、カラー、フォント、共通コンポーネント）
2. トップページ（ヒーロー、信頼性セクション、おすすめ商品）
3. 商品一覧（カード刷新、フィルタ、ソート）
4. 商品詳細（画像ギャラリー、CTA最適化、関連商品）
5. カート（ドロワー式、UX最適化）
6. ヘッダー/フッター（sticky blur、モバイル対応）
7. アニメーション & マイクロインタラクション
8. レスポンシブ & パフォーマンス最適化

## 必須ルール

- `tailwindplus/` テンプレートを実装前に必ずレビューしベースとして使用
- テンプレートのコピペではなくさらなる高みを目指す
- 迷ったら apple.com / stripe.com / airbnb.com を参考に
- 日本語 line-height: 1.8、タッチターゲット 44x44px以上
- 各フェーズ完了時に `pnpm -C apps/storefront build` で確認
- 目標: 「このサイトで買い物したい」と直感的に感じるデザイン
